Legend

Colors go from darker to lighter signifying the importance of sessions (according to me, at least): Cours > TD > TP @Where A or Amphi: Cours S or Salle: TD L or Lab: TP Who G: Groupe Sec: Section Modules: BD: Bases de données RC: Réseaux de télécommunication SE: Systèmes d'exploitation 1 DAW: Développement d'applications Web GL: Génie logiciel 1 TG: Théorie des graphes AJEL: Aspects juridiques et économiques des logiciels

What?

- The filter basically takes a CSS selector and adds/removes classes from divs to do its job. - In case an invalid filter is provided, it highlights all cells. - `edt.htm` is `EDT_S4_16-2-2019-sansTPBDG5.pdf` as exported by MS Word 2016. When a new schedule is published, only the 'edt.htm' need to be replaced. - It must be run on a server cause it uses `fetch` and it doesn't work on the 'file:' protocol. - Tested using Google Chrome v71. Developed using Notepad++ and lighttpd. - It uses latest JavaScript features like Array.prototype.flatMap so the code probably needs to be transpiled to support relatively older browsers. - Inspired by an interesting script of my friend Wanis.
